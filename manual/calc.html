calc(1) - LynxBot manual

NAME

calc - perform basic calculations

SYNOPSIS

$calc EXPRESSION

DESCRIPTION

calc evaluates the mathematical expression given by EXPRESSION and prints the result.

EXPRESSION can contain numbers, parentheses and the mathematical operators +, -, *, / and ^.

EXAMPLES

Evaluates the expression:

$calc (4 + 7) ^ (3 / 2) * (-1.5)
