help(1) - LynxBot manual

NAME

help - view command reference manuals

SYNOPSIS

$help CMD

DESCRIPTION

help posts a link to the LynxBot manual page for the bot command CMD.

EXAMPLES

Posts link to this page:

$help help

Posts link to setrec manual page:

$help setrec
