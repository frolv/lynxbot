config(4) - LynxBot manual

DESCRIPTION

Most of LynxBot's configuration is done within the config file found in the LynxBot config directory.

Settings in the config file are given as key-value pairs separated by a =, with one setting per line.

Lines beginning with a # are comments which are ignored by the parser.

SECTIONS

There are 6 sections within the config file. Click one below for detailed information about the settings within that section:

  1. General settings
  2. Moderation settings
  3. Whitelisted websites
  4. Giveaway settings
  5. Recurring messages
  6. Other settings
