Twitch Authorization(3)
- LynxBot manual

DESCRIPTION

Authorizing LynxBot with your Twitch account allows it to edit your channel's status through the status command.

USAGE

When LynxBot is first launched, it will run a setup process in which it provides instructions on how to authorize it with Twitch.

If you chose not to do the setup, you can authorize LynxBot by going to the following authorization URL, signing in with your Twitch account and entering the generated code into the LynxBot config file, in the twitchtok field.
