manual(1) - LynxBot manual

NAME

manual - view the lynxbot manual

SYNOPSIS

$manual

DESCRIPTION

manual posts the URL of the LynxBot manual index page.

commands is an alias for the manual command.

OPTIONS

--help
print brief description and usage information

EXAMPLES

Posts link to manual index:

$manual
