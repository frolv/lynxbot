Selection Wheel(3)
- LynxBot manual

DESCRIPTION

The Selection Wheel is a feature of LynxBot which allows it to randomly select items from various categories.

Each Twitch user can be assigned one item from a single category once every 20 minutes.

USAGE

The implementation of the Selection Wheel is defined in json/wheen.json.

The wheel is activated by typing the keyword specified by wheelcmd followed by the name of one of the categories in categories, or the word check.

EXAMPLES

Using the default examples.json file, this would assign the user either 1, 2 or 3:

$swheel number
