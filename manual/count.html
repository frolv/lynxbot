count(2) - LynxBot manual

NAME

count - manage message counts

SYNOPSIS

$count COMMAND

DESCRIPTION

count begins, ends and views the results of message counts.

OPTIONS

--help
print brief description and usage information

COMMANDS

start
begin a session of message counting
stop
end the current count
display
show the top 10 results of the previous count and number of votes in descending order

EXAMPLES

Starts a new count:

$count start

Stops the active count:

$count stop

Shows results from last count:

$count display

SEE ALSO

strawpoll(2), Message Counting(3)
