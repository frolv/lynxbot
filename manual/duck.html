duck(1) - LynxBot manual

NAME

duck - search duckduckgo with a query string

SYNOPSIS

$duck QUERY

DESCRIPTION

duck searches DuckDuckGo with the string QUERY.

Including DuckDuckGo bangs in the query allows LynxBot to directly link to pages on thousands of different websites.

OPTIONS

--help
print brief description and usage information

EXAMPLES

Searches DuckDuckGo for Old School Runescape:

$duck old school runescape

Links to the OSRS wiki page for Construction (using DDG bangs):

$duck construction !osrs
