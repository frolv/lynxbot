configdir(4) - LynxBot manual

DESCRIPTION

The LynxBot configdir is the directory which stores all of LynxBot's configuration files.

It contains the subdirectory json which stores all the json files.

LOCATION

On GNU/Linux: ~/.lynxbot/

On Windows: %LYNXBOT_INSTALL_DIR%
