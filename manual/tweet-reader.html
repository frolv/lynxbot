Tweet Reader(3)
- LynxBot manual

DESCRIPTION

LynxBot has a Tweet Reader which identifies links to Twitter statuses in the chat and responds by posting the contents of the Tweet.

USAGE

The Tweet Reader requires LynxBot to be authenticated with a Twitter account.

When LynxBot is first launched, it will enter an interactive Twitter authentication setup in which it provides instructions on how to authenticate it.

The interactive setup can be repeated by deleting the twitter file from the LynxBot configuration directory.

Note that Twitter authentication codes are currently stored in plaintext.
