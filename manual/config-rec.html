recurring(4) - LynxBot manual

DESCRIPTION

The Recurring messages section of the LynxBot config file stores the bot's recurring messages.

SETTINGS

All recurring messages are stored in the recurring list.

SYNOPSIS

Each item in the list has two settings: a period and a message.

The period is the interval, in minutes, at which the message is posted.

The message is the string to post in the chat.
