editcom(2) - LynxBot manual

NAME

editcom - modify a custom command

SYNOPSIS

$editcom [-c COOLDOWN] COMMAND [RESPONSE]

DESCRIPTION

editcom changes the cooldown or response of a custom command.

COMMAND is the keyword that activates the command, without the leading dollar sign $.

At least one of COOLDOWN or RESPONSE must be given.

OPTIONS

-c COOLDOWN
set the cooldown of the command to COOLDOWN seconds

EXAMPLES

Changes the response of the command $hi to "hey wassup hello"

$editcom hi hey wassup hello

Changes the cooldown of the command $hi to 60 seconds:

$editcom -c 60 hi

Changes the response of the command $hi to "hey wassup hello" and the cooldown to 60 seconds:

$editcom -c 60 hi hey wassup hello
