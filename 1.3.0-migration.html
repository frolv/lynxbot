LynxBot v.1.3.0 Migration

LynxBot v1.3.0 greatly changes bot configuration by combining most of the old configuration files into one larger, centralized file.

Due to this, if you are upgrading to v1.3.0 from a previous version, you will have to modify your current configuration files to match the new format.

Simply follow the steps below to complete this process:

  1. Extract the new config file to your LynxBot config directory.
  2. Copy the necessary data from the old settings, extra8ballresponses, submessage, recurring, giveaway/giveaway-settings and whitelist files into config (pay attention to the new format - see config(4)).
  3. Move the file giveaway/giveaway into the main config directory.
  4. Delete the settings, extra8ballresponses, submessage, recurring and whitelist files and the giveaway directory.

After this, LynxBot will be set up to use the new configuration file, which you can then edit to your liking.
