submit(1) - LynxBot manual

NAME

submit - submit a message to the streamer

SYNOPSIS

$submit MSG

DESCRIPTION

submit submits the string MSG to the streamer. The username of the person who submits a message is saved alongside it.

OPTIONS

--help
print brief description and usage information

EXAMPLES

Submits the message "Hello world":

$submit Hello world
