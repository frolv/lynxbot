showrec(2) - LynxBot manual

NAME

showrec - show recurring messages

SYNOPSIS

$showrec [ID]

DESCRIPTION

showrec prints the specified recurring message(s) with their intervals.

OPTIONS

--help
print brief description and usage information

EXAMPLES

Shows all recurring messages:

$showrec

Shows the second recurring message:

$showrec 2
