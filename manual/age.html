age(1) - LynxBot manual

NAME

age - check length of channel relationships

SYNOPSIS

$age <-f|-s>

DESCRIPTION

age checks how long a user has been following or subscribed to the bot's channel.

OPTIONS

-f, --follow
check how long user has been following channel
--help
print brief description and usage information
-s, --sub
check how long user has been subscribed to channel

EXAMPLES

Shows how long you have been following the channel:

$age -f

Shows how long you have been subscribed to the channel:

$age --sub
