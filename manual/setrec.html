setrec(2) - LynxBot manual

NAME

setrec - enable or disable recurring messages

SYNOPSIS

$setrec COMMAND

DESCRIPTION

setrec activates and deactivates the broadcasting of recurring messages.

OPTIONS

--help
print brief description and usage information

COMMANDS

on
enables recurring messages
off
disables recurring messages

EXAMPLES

Turns recurring messages on:

$setrec on

Turns recurring messages off:

$setrec off
