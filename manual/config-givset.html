giveaway-settings(4) - LynxBot manual

DESCRIPTION

The Giveaway settings section of the LynxBot config file contains all the settings that govern bot giveaways.

SETTINGS

The settings within the Giveaway settings section are as follows:

giveaway_active

Whether giveaways are active or inactive.

follower_giveaway

Whether follower giveaways should occur.

follower_limit

The number of followers required for a follower giveaway to occur.

timed_giveaway

Whether time based giveaways should occur.

time_interval

The average period, in minutes, at which timed giveaways occur.
