LynxBot: A bot for Old School Runescape Twitch chat

LynxBot is a Twitch.tv IRC bot designed specifically for use on Old School Runescape streams. It offers a number of commands that assist Runescape streams, such as Grand Exchange and Hiscore lookups, as well as the flexibility of customization through user defined commands, automated responses and more.

For setup instructions, see setup.

See the wiki for full documentation on LynxBot's functionality.
